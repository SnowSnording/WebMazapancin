Politica de privacidad.

Mazapancin unicamente recopila el ID de su servirdor y lo guarda en un base de datos privada, esto para poder guardar y cargar los cambios que se le hagan al bot en el
respectivo servidor, esta informacion unicamente es accesible para el desarollador y no se comparte con terceros. Al ser un cliente de Discord no usamos coockies ni similares
para proporcionarte anuncios o similares. El bot usa datos que la misma API de discord utiliza, los cuales se encientran en los terminos y condiciones, estos datos no se
recopilan ni se guardan de ninguna manera en bases de datos desarolladas por SnowSnording o terceros. Nos reservamos el derecho de cambiar esta Politica de privacidad
cuando lo veamos necesario sin dar un aviso previo.
